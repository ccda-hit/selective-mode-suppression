# Selective Mode Suppression (SMS) for fNIRS Motion Artifact Removal

This repository will contain the official implementation of the **Selective Mode Suppression (SMS)** algorithm, a novel targeted approach for motion artifact suppression in fNIRS data utilizing Variational Mode Decomposition (VMD).

This code is part of the implementation for the paper:  
**"Removal of Motion Artifacts in fNIRS using Selective Mode Suppression"**

## Code Release
The complete Python/MATLAB source code and installation instructions will be made publicly available in this repository immediately following the formal peer-review acceptance and publication of the paper.

For any immediate inquiries, please contact the Center for Complex Data Analysis (CCDA) at HIT.
